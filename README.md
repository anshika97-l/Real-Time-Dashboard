# 🌦 Real-Time Weather Dashboard

A real-time weather monitoring dashboard built using Python, Streamlit, and OpenWeather API.

## Features

* Real-time weather data
* Temperature, Humidity, and Pressure metrics
* Interactive weather analytics chart
* Secure API key management using `.env`
* Responsive Streamlit dashboard

## Technologies Used

* Python
* Streamlit
* Pandas
* Requests
* OpenWeather API

## Installation

1. Clone the repository
2. Install dependencies

```bash
pip install -r requirements.txt
```

3. Create a `.env` file

```env
OPENWEATHER_API_KEY=your_api_key_here
```

4. Run the application

```bash
streamlit run app.py
```

## Project Output

Displays live weather information for any city entered by the user along with graphical weather analytics.
